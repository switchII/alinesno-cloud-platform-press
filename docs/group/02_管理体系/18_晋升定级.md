## 概述

此文档包括 中台部门工程师的岗位职级定级划分，晋升

### 试行阶段

此文档包括 Java 部门工程师的岗位职级定级划分，晋升，考核标准等

### 等级职责

等级职责的说明为了更便于管理和人员培养，等级职责按人员等级划分，与外部等级职责无关联，便于内部人员成长，工作向上向下进行交待，
汇报以周报形式进行汇报

### 内部等级晋升

由晋升人员发起，向技术委员会进行述职答辩晋升等级，发起时间，由团队管理制度决定。
拥有相关证书为加分项，比如高级工程师、AWS 架构师、项目管理等等。
