# 项目管理

## 概述

项目管理是项目管理人员对项目的工作、进度、任务、流程、人员等进行的管理监控，主要由项目经理承担，质量管控人员辅助。Java 部门的项目管理整体流程基本是在禅道上进行的，可以在禅道上进行相应的任务指派、工作分解、过程跟进及结果检查等。

## 项目交付流程
