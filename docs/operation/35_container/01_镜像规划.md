# 容器镜像规划

## 概述

基础容器镜像用于平台快速部署和管理，用于更有优势的部署，同时更统一的环境，
此处容器镜像采用 docker，进行统一管理，此处的容器规范的是基础平台业务的

```
alinesno-vision/x86/Dockerfile

```

```
deploy/alinesno-container-base-centos_x64/k8s-single/
```
