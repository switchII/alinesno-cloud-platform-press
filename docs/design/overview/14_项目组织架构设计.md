# 项目组织管理规划

## 概述

项目规划管理涉及到整个项目的生命周期，即从项目的立项、范围、进度、成本、质量、资源、风险、合同、变更等等一系统的文档产物。
项目文档的一些基础管理规划，形成原子性管理
