# IoT 整体技术构架

### 概要

> 此建立在研发中台架构之上

设备连接物联网平台，与物联网平台进行数据通信。物联网平台可将设备数据流转到其他阿里云产品中进行存储和处理。这是构建物联网应用的基础。
此处定义整体技术架构选型
