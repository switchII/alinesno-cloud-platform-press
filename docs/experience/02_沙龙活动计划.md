# 活动计划

## 概述

沙龙活动是进一步的进行团队组织的活动，为线下或者线上的活动，主要是为了更好的进行产品能力分享，技术能力分享，
经验分享等，进一步的活跃社区组织，进一步的加强社区和外部的沟通交流，进一步的提升产品能力和吸收专业人员的
建议和意见。
